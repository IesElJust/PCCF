# Contribució dels RA a les competències professionals

<!-- 
De la guía:
-->

Tenint com a referent màxim la globalitat de les competències professionals, personals i socials associades al cicle formatiu, programem de manera competencial, prenent els Resultats d'Aprenentatge (RA) com a eix central. Els continguts esdevenen mitjans per a desenvolupar aquestes competències, mentre que els Criteris d'Avaluació (CA) ens permeten mesurar-ne el grau d'assoliment.

Tant els RA com els CA es defineixen al Reial decret del títol i es contextualitzen dins del Projecte Curricular del Cicle Formatiu (PCCF). En aquest document es recull la taula següent, que mostra la contribució dels diferents mòduls a les competències professionals:

Veiem de manera explícita aquesta relació:

**RA1. Instal·la sistemes operatius, analitzant les seues característiques i interpretant la documentació tècnica.**: Aquest RA contribueix a le competències:

    * *1. Administrar sistemes operatius de servidor, instal·lant i configurant el programari, en condicions de qualitat per a assegurar el funcionament del sistema.*
    * *5. Optimitzar el rendiment del sistema configurant el dispositiu maquinari d'acord amb els requisits de funcionament.*
    * *6. Avaluar el rendiment del dispositiu maquinari identificant possibilitats de millores segons les necessitats de funcionament.*

    

**RA2. Configura el programari de base, analitzant les necessitats d'explotació del sistema informàtic.**: Aquest RA contribueix a le competències:

    * *1. Administrar sistemes operatius de servidor, instal·lant i configurant el programari, en condicions de qualitat per a assegurar el funcionament del sistema.*
    * *3. Administrar aplicacions instal·lant i configurant el programari, en condicions de qualitat per a respondre a les necessitats de l'organització.*
    * *5. Optimitzar el rendiment del sistema configurant el dispositiu maquinari d'acord amb els requisits de funcionament.*

**RA3. Assegura la informació del sistema, descrivint els procediments i utilitzant còpies de seguretat i sistemes tolerants a fallades.**: Aquest RA contribueix a le competències:

    * *9. Implementar solucions d'alta disponibilitat, analitzant les diferents opcions del mercat, per a protegir i recuperar el sistema davant situacions imprevistes.*
    * *10. Supervisar la seguretat física segons especificacions del fabricador i el pla de seguretat per a evitar interrupcions en la prestació de servicis del sistema.*
    * *11. Assegurar el sistema i les dades segons les necessitats d'ús i les condicions de seguretat establides per a previndre fallades i atacs externs.*
    * *13. Diagnosticar les disfuncions del sistema i adoptar les mesures correctives per a restablir la seua funcionalitat.*
  
**RA4. Centralitza la informació en servidors administrant estructures de dominis i analitzant els seus avantatges.**: Aquest RA contribueix a le competències:

    * *1. Administrar sistemes operatius de servidor, instal·lant i configurant el programari, en condicions de qualitat per a assegurar el funcionament del sistema.*
    * *12. Administrar usuaris d'acord amb les especificacions d'explotació per a garantir els accessos i la disponibilitat dels recursos del sistema.*

**RA5. Administra l'accés a dominis analitzant i respectant requeriments de seguretat.**: Aquest RA contribueix a la competència:

    * *11. Assegurar el sistema i les dades segons les necessitats d'ús i les condicions de seguretat establides per a previndre fallades i atacs externs.*
    * *12. Administrar usuaris d'acord amb les especificacions d'explotació per a garantir els accessos i la disponibilitat dels recursos del sistema.*

**RA6. Detecta problemes de rendiment, monitorant el sistema amb les ferramentes adequades i documentant el procediment.**: Aquest RA contribueix a la competència:

    * *5. Optimitzar el rendiment del sistema configurant el dispositiu maquinari d'acord amb els requisits de funcionament.*
    * *6. Avaluar el rendiment del dispositiu maquinari identificant possibilitats de millores segons les necessitats de funcionament.*
    * *13. Diagnosticar les disfuncions del sistema i adoptar les mesures correctives per a restablir la seua funcionalitat.*
    * *14. Gestionar i/o realitzar el manteniment dels recursos de la seua àrea (programant i verificant el seu compliment), en funció de les càrregues de treball i el pla de manteniment.*

**RA7. Audita la utilització i accés a recursos, identificant i respectant les necessitats de seguretat del sistema.**: Aquest RA contribueix a le competències:

    * *10. Supervisar la seguretat física segons especificacions del fabricador i el pla de seguretat per a evitar interrupcions en la prestació de servicis del sistema.*
    * *11. Assegurar el sistema i les dades segons les necessitats d'ús i les condicions de seguretat establides per a previndre fallades i atacs externs.*
    * *12. Administrar usuaris d'acord amb les especificacions d'explotació per a garantir els accessos i la disponibilitat dels recursos del sistema.*

**RA8. Implanta programari específic amb estructura client/servidor donant resposta als requisits funcionals.**: Aquest RA contribueix a le competències:

    * *2. Administrar servicis de xarxa (web, missatgeria electrònica i transferència d'arxius, entre altres) instal·lant i configurant el programari, en condicions de qualitat.*
    * *3. Administrar aplicacions instal·lant i configurant el programari, en condicions de qualitat per a respondre a les necessitats de l'organització.*
